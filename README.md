# About

Это тестовая гит-репа для обучения гиту !

## test-py

Этот скрипт предназначен для подключения к серверу

Создать файл `config.json`, например, с таким содержимым:

```json
{
  "ip": "1.2.3.4",
  "login": "your_login",
  "password": "your_password"
}
```

Запустить скрипт: `python3 test.py`

# Chess-bot

Шахматный бот использующий движок stockfish.

## Install

* Установить stockfish: `apt install stockfish`
* Создать venv ` python3 -m venv venv-chess`
* Активировать venv `source venv-chess/bin/activate`
* Установить зависимости `pip install -r requirements.txt`

## Usage

* Запустить бота `python3 chess-bot.py`
* До начала партии установить настройки в окне бота:
    * Цвет ваших фигур: белые/чёрные
    * Режим работы бота:
        * **Лучший ход** - бот будет только подсказывать лучший ход
        * **Автоход** - бот будет сам двигать ваши фигуры (надо в браузере перейти на сайт <https://www.chess.com>
          и развернуть окно на весь экран)

### Играем партию

Во время партии записывайте каждый ход противника в поле ввода, например: `c7c6` или `e2 e4`.

Бот отправит расположение фигур на доске в программу stockfish и получит рекомендацию лучшего хода,
выведет её в поле "Лог действий" и будет считать, что вы сходите именно так.

Если выбран режим "автоход", то бот сам передвинет фигуру в окне браузера.
В режиме "лучший ход" - передвигайте фигуры самостоятельно
